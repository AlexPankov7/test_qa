Краткая инструкция по работе с GIT.
_______________________________________

1.Создаём репозиторий на github
2.Клонируем его с помощью SSH key or HTTPS
3.Переходим в папку в которую нужно склонировать проект(команда cd)
4.Чтобы склонировать вводим команду git clone и далее SSH key (именно публичный ключ!!!)
   (Команда вставить в git bush - SHIFT + INSERT)

Далее git bush спросит разрешение к подключению к хосту github.com. Вводим yes
Далее будет запрос пароля. Вводим его.

Можем убедиться что создалась директория tast_qa  и если мы в неё перейдём (cd test_qa/)то увидим флажёк master или main.

И чтобы убедиться что это именно github а не какой-то другой хостинг введём команду
git remote -v


Следующее что нужно сделать это создать файл и поместить информацию в этот файл.
1.Создаём файл (это файл song.txt) и делаем в нём изменения.
2.Возвращаемся в git bush и вызываем команду git status. После чего git скажет нам о том что в текущей папке появился новый файл song.txt и предложит добавить его в коммит(add .)
3.Добавляем файл в коммит командой git add song.txt
4.Вызываем git status ещё раз и видим что song.txt подсвечивается зелёным. 
5.Делаем непосредственно коммит git commit -m "тут прописываем название нашего коммита"
6.Команда git log покажет историю коммитов.
7.И осталось только синхронизировать свой локальный репозиторий с удалённым (залить все изменения на github) команда git push
